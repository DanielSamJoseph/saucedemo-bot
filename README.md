# saucedemo-bot
A side project built with Selenium to experiment with automation concepts and improve my skills.

If you wish to run this code then please do the following steps:

1)This is the link for donwnloading the driver:https://googlechromelabs.github.io/chrome-for-testing/#stable

2)Download the appropriate ChromeDriver version from the official ChromeDriver download page, matching your installed Google Chrome version.

3)Extract the downloaded ZIP file and locate the chromedriver.exe file and install it.

4)Place chromedriver.exe in a directory that is included in your system's PATH environment variable, such as C:\Windows, or specify the full path to the executable when initializing Selenium in your code

5)Then go to your terminal and write "pip install selenium" to install selenium then your done and run the code.
